# Practice on Logical Operators
## There are three major logical operators in C language

* Logical NOT (!)
* Logical OR (||)
* Logical AND (&&)

### And Operators

* True and True is True
* True and False is False
* F T -> F
* F F -> F

### Or Operators

* T T -> T
* T F -> T
* F T -> T
* F F -> F
